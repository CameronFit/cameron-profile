What you see

When you select a country, the city dropdown immediately updates with cities from that country. The summary text changes as you make selections.

What's under the hood

Angular signals and computed() make one field react to changes in another. No manual event handling or state synchronization needed. The reusable SelectInputComponent uses ControlValueAccessor for clean integration.
